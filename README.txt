Banking system (Java project)
Project Title: BankApp 

This Java project simulates a simple banking system.
Users can create accounts, log in, deposit money, withdraw, check balance, and use additional features like buying tickets, taking loans, etc.

Our banking system have base features and some additional, there all features:
	•	Create new account
	•	Login system
	•	PIN code check
	•	Deposit / Withdraw
	•	Check balance
	•	Buy movie tickets
	•	Take credit (loan)
	•	Simple menu navigation  So how run our project? Firstly
1)Open project in IntelliJ (or any java reader)
2)Make sure your code is inside the src folder.
3)Find and run the file:Main.java or BankAPP.java
4)Write to the console to use menu.
